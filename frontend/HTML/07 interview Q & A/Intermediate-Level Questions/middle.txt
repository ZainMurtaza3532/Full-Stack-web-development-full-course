7. What is the difference between id and class attributes?

Answer:
id: Unique identifier for one element (#id in CSS).
class: Can be used for multiple elements (.class in CSS).

8. What is the difference between <link> and <a> tags?

Answer:
<a>: Defines a hyperlink (<a href="page.html">Click</a>).
<link>: Defines a relationship between a document and an external resource (commonly for CSS: <link rel="stylesheet" href="style.css">).

10. What is the difference between local storage and session storage in HTML5?

Answer:
Local Storage: Stores data with no expiration (persists after browser close).
Session Storage: Stores data only for the session (cleared when tab/browser closes).
