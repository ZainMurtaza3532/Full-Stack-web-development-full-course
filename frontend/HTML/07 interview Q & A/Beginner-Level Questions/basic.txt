1. What is HTML?

Answer:
HTML (HyperText Markup Language) is the standard language used to create web pages. It defines the structure of a webpage using elements like headings, paragraphs, images, links, tables, and forms.

----------------------------------------------------------------

2. What are tags and attributes in HTML?

Answer:
Tags define elements in HTML (e.g., <p>, <h1>, <img>).
Attributes provide extra information about an element (e.g., <img src="image.jpg" alt="description">).

----------------------------------------------------------------

3. What is the difference between block-level and inline elements?

Answer:
Block-level elements: Start on a new line and take full width. Example: <div>, <p>, <h1>.
Inline elements: Stay in line with text and only take required space. Example: <span>, <a>, <strong>.

----------------------------------------------------------------

4. What is the difference between HTML and HTML5?

Answer:
HTML5 introduced new semantic elements (<header>, <footer>, <article>, <section>).
Supports audio & video (<audio>, <video>).
Supports local storage & session storage.
Provides canvas for drawing graphics.
Improved support for mobile devices.

----------------------------------------------------------------

5. What is DOCTYPE in HTML?

Answer:
<!DOCTYPE html> tells the browser that the document type is HTML5. It ensures the page is rendered correctly.
